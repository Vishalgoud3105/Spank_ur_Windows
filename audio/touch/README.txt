# audio/touch/
# Gesture sound categories for trackpad and touchscreen (spank.py v5)
#
# 1_single/   Single finger gentle touch  ? soft / breathing / whisper sounds
# 2_double/   Two finger touch            ? gentle moan sounds
# 3_pinch/    Two finger pinch/squeeze    ? squeeze / gasp sounds
# 4_triple/   Three fingers              ? deeper moan sounds
# 5_palm/     Four+ fingers or palm      ? intense grab / gasp sounds
#
# Add any MP3 / WAV / OGG files to each folder.
# Files are picked randomly with no back-to-back repeats.
# The more files you add, the more variety you get.
#
# These are SEPARATE from the pain/sexy/halo sounds.
# Gesture sounds play on their own audio channel so they
# can overlap with the main slap sounds simultaneously.
